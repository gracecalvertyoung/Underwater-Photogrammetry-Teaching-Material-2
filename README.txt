This repository contains teaching material for learning PhotoScan and Rhinoceros 3D, two softwares critical to the method for creating and analysing 3D models described in the following publication: 

1. Young GC, Dey S, Rogers AD, Exton D. Cost and time-effective method for multi-scale measures of rugosity, fractal dimension, and vector dispersion from coral reef 3D models. PLoS One. 2017;12: e0175341. doi:10.1371/journal.pone.0175341

START WITH the file “Lesson Notes.pdf” It contains references to the other files in this repository.
